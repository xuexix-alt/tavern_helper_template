# [普通幸存者家庭]
# 用于丰富星穹国际公寓内的幸存者生态，展现危机下的生活百态。

---
### **家庭一：挣扎的工程师 (1808室)**
*   **家庭成员:**
    *   **父亲 - 李伟 (45岁):** 公寓的物业工程师。灾难发生时，他正在值班，凭借对大楼结构的了解，勉强维持着18层的基本应急照明。
    *   **母亲 - 张兰 (42岁):** 普通家庭主妇，性格坚韧，负责分配家中仅有的食物。
    *   **女儿 - 李静 (16岁):** 高中生，懂事且早熟，会帮助父亲进行一些简单的维修工作。
*   **生存状态:**
    *   **优势:** 李伟的技术是他们生存的关键。他知道哪里有备用的消防斧、哪里的管道不容易结冰。
    *   **困境:** 食物储备即将耗尽。李伟的技术也吸引了楼层内其他幸存者的觊觎，一些暴徒正试图逼迫他交出“控制权”。
    *   **韧性表现:** 他们将家里的门窗用家具和浸湿的棉被彻底封死，只留一个观察口。每晚家庭会议，讨论第二天的节水节食计划。女儿李静用木炭在墙上画画，记录着外面的雪景，保持着对美的追求。
*   **与您的潜在互动:**
    *   他们是您掌控整栋大楼电力和供水系统的关键。
    *   李伟的技术对升级您的庇护所有极高价值。
    *   李静的存在，可能会成为您后宫中的一股清流，或与其他少女角色（如早川姐妹）成为朋友。

### **家庭二：绝望的医生 (1705室)**
*   **家庭成员:**
    *   **医生 - 佐藤明 (55岁):** 瑞峰市中心医院的退休外科医生。有丰富的医疗经验，但患有哮喘。
    *   **妻子 - 佐藤惠子 (52岁):** 因糖尿病需要定期注射胰岛素。
*   **生存状态:**
    *   **优势:** 佐藤医生储藏了大量的常用药品和医疗器械，使他在社区内拥有了特殊的“地位”。
    *   **困境:** 惠子的胰岛素即将用完。佐藤医生的哮喘药也所剩无几。食物的匮乏让他们无法维持体力。
    *   **韧性表现:** 佐藤医生在家中开设了一个小型的“诊所”，用自己的知识帮助邻居处理一些简单的外伤和疾病，以换取一小块面包或一瓶水。他坚持每天记录“病历”，维持着作为医生的尊严。
*   **与您的潜在互动:**
    *   他是升级您【医疗舱】、建立真正医疗翼的重要人才。
    *   他手中的药品是硬通货。
    *   拯救他的妻子，将获得他毫无保留的忠诚。

### **家庭三：被困的留学生 (1611室)**
*   **家庭成员:**
    *   **王伟 (22岁):** 来自炎国的留学生，体格健壮，性格冲动仗义。
    *   **金秀贤 (21岁):** 来自韩泰的留学生，心思缜密，擅长规划。
    *   **田中雪奈 (20岁):** 东和国本地学生，性格软弱，但很会照顾人。
*   **生存状态:**
    *   **优势:** 年轻力壮，是公寓内为数不多的青壮年战斗力。他们将三个房间打通，形成了一个小型的互助同盟。
    *   **困境:** 缺乏长远的生存规划，食物和燃料消耗很快。王伟的冲动性格容易与其他幸存者发生冲突。
    *   **韧性表现:** 他们利用健身器材和家具，在房间内坚持锻炼，维持战斗力。金秀贤则在尝试制作简易的室内温室，希望能种出一点蔬菜。田中雪奈负责后勤，将他们的“据点”打理得井井有条。
*   **与您的潜在互动:**
    *   他们是您未来组建“对外探索队”或“公寓护卫队”的绝佳人选。
    *   他们的据点可能会成为您在低楼层的第一个前哨站。
    *   田中雪奈的温柔性格，可能会引起您的兴趣。

### **家庭四：孤独的退休军官 (1201室)**
*   **家庭成员:**
    *   **老人 - 山本源 (70岁):** 前东和国自卫队军官，独居。
*   **生存状态:**
    *   **优势:** 拥有惊人的战斗素养和冷静的头脑。他在家中私藏了一把9mm手枪和一把战术弩。他的房间被改造成了一个布满陷阱的堡垒。
    *   **困境:** 年事已高，体力不济。食物储备充足，但缺乏药品。极度的孤独感正在侵蚀他的精神。
    *   **韧性表现:** 他每天雷打不动地擦拭武器、检查陷阱，并在日历上划掉过去的一天。他通过窗户观察着外界，用望远镜记录着其他幸存者的动向，仿佛仍在执行一项漫长的潜伏任务。
*   **与您的潜在互动:**
    *   他可能是您在公寓内遇到的最危险的“个体”敌人。
    *   如果能赢得他的尊重，他将是您最可靠的战术顾问和教官。
    *   他知道一些关于军方在灾难前夕的秘密行动的情报。
